emacs-bootstrap
===============

Quick setup GNU Emacs.

#Install

Use `make install`.

After install in emacs `M-x irony-install-server`

#Use server

Run `emacsclient -a ""` with `-c` for GUI mode and `-t` for TTY mode.
For set `EDITOR` use string `export EDITOR=emacsclient\ -a\ \"\"\ -t` in your profile.
